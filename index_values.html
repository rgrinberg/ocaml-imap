Index of values


A
append [Imap]
append m flags id data appends data as a new message to the end of the mailbox m.

C
capability [Imap]
capability returns the list of capabilities supported by the server.
check [Imap]
check requests a checkpoint of the currently selected mailbox.
close [Imap]
close permanently removes all messages that have the `Deleted Imap.flag set from the currently selected mailbox, and returns to the authenticated state from the selected state.
connection [Imap]
connection () creates a new connection object.
copy [Imap]
copy uid set m copies the messages in set to the end of the specified mailbox m.
create [Imap]
create m creates a mailbox named m.

D
dst [Imap]
dst c s i l provides c with l bytes of output storage in s starting at i.
dst_rem [Imap]
dst_rem c is the number of bytes still available for output in the current output buffer of c.

E
enable [Imap]
examine [Imap]
examine condstore m is identical to select condstore m and returns the same output; however, the selected mailbox is identified as read-only.
expunge [Imap]
expunge permanently removes all messages that have the `Deleted Imap.flag set from the currently selected mailbox.

F
fetch [Imap]
fetch uid changed vanished set att retrieves data associated with the message set set in the current mailbox.
fetch_all [Imap]
fetch_all uid changed vanished set is equivalent to fetch uid changed vanished set a, where a = [`Flags; `Internal_date; `Rfc822_size; `Envelope].
fetch_fast [Imap]
fetch_fast uid changed vanished set is equivalent to fetch uid changed vanished set a, where a = [`Flags; `Internal_date; `Rfc822_size].
fetch_full [Imap]
fetch_full u c v s is equivalent to fetch u c v s a where a = [`Flags; `Internal_date; `Rfc822_size; `Envelope; `Body].

L
list [Imap]
list ref m returns a subset of names from the complete set of all names available to the client.
login [Imap]
login user pass identifies the client to the server and carries the plaintext password authenticating this user with password pass.
logout [Imap]
logout gracefully terminates a session.
lsub [Imap]
lsub ref m is identical to Imap.list, except that it returns a subset of names from the set of names that the user has declared as being "active" or "subscribed".

N
noop [Imap]
noop does nothing.

P
plain [Imap]
plain user pass authenticates via PLAIN mechanism using username user and password pass.
pp_error [Imap]
pp_response [Imap]

R
rename [Imap]
rename oldname newname command changes the name of a mailbox from oldname to newname.
run [Imap]
run c v performs v on the connection c.

S
search [Imap]
search uid sk searches the mailbox for messages that match the given searching criteria.
select [Imap]
select condstore m selects the mailbox m so that its messages can be accessed.
src [Imap]
src c s i l provides c with input taken from l bytes from s starting at i.
status [Imap]
status requests status information of the indicated mailbox.
store_add_flags [Imap]
store_add_flags uid silent unchanged set flags adds flags flags to the message set set.
store_add_labels [Imap]
store_add_labels is like Imap.store_add_flags but adds Gmail labels instead of regular flags.
store_remove_flags [Imap]
store_remove_flags is like Imap.store_add_flags but removes flags instead of adding them.
store_remove_labels [Imap]
store_remove_labels is like Imap.store_add_labels but removes labels instead of adding them.
store_set_flags [Imap]
store_set_flags is like Imap.store_add_flags but replaces the set of flags instead of adding to it.
store_set_labels [Imap]
store_set_labels is like Imap.store_add_labels but replaces the set of labels instead of adding to it.
subscribe [Imap]
subscribe m adds the mailbox m to the server's set of "active" or "subscribed" mailboxes as returned by the Imap.lsub command.

U
unsubscribe [Imap]
unsubcribe m removes the mailbox m from the server's set of "active" or "subscribed" mailboxes as returned by the Imap.lsub command.

X
xoauth2 [Imap]
xoauth2 user token authenticates via XOAUTH2 mechanishm user username user and access token token.
